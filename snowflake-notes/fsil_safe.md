Fail-safe
📌 Definition

Fail-safe is a non-configurable recovery mechanism provided by Snowflake after Time Travel expires.

⏱ Duration
Fixed: 7 days
⚠️ Important Notes
Cannot query data during Fail-safe
Only Snowflake support can restore data
Used for disaster recovery
🔄 Data Lifecycle
Active Data
Time Travel Period
Fail-safe Period (7 days)
Permanent Deletion
