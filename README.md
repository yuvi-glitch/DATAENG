# SQL Practice — Snowflake

Personal SQL learning repository. All queries written and tested in Snowflake.

---

## About

This repository contains SQL practice queries, problem solutions, and notes written while learning SQL for data engineering roles. All queries are written in Snowflake SQL syntax and tested on a custom `salesdb` dataset.



## Tech stack
- Snowflake
- SQL
- SnowPro Core Certified

## Dataset

A custom `salesdb` database with the following tables:
- `customers` — customer details and scores
- `employees` — employee info, departments, salaries
- `products` — product catalogue with categories and pricing
- `orders` — order transactions with status and sales
- `orders_archive` — historical order data
